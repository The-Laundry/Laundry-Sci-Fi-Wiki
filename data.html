📂 Data Folder

🤖 AI (Infinite Wiki)

Connect an OpenAI-compatible endpoint to generate new articles from broken wikilinks, auto-write summaries, and build an embedding index for context search. The API key is stored only in your browser's localStorage — it is never written to settings.json or included in exports.

Export All Data

Download a JSON file containing all articles, categories, timelines, events, and settings. Works as a universal backup regardless of storage mode.

Import Data

Load a previously exported JSON file. Warning: this replaces all current data.

Export Single Article

Export one article as a standalone JSON file.


Clear All Data

Permanently delete all articles, categories, timelines, and events. This cannot be undone.
