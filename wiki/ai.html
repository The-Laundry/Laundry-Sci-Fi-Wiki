AI Settings

🤖 Connection & Behavior

Connect an OpenAI-compatible endpoint to power article generation, summary writing, and semantic search. The API key is stored only in your browser's localStorage — it is never written to settings.json or included in exports.

✏️ Prompts

These prompts drive every AI interaction. Edit them to tune tone, format, or worldbuilding rules. Leave a field blank (or identical to the default) to restore it. Placeholders such as {title} and {content} are filled in at runtime.

Article generation — system prompt
Main behavioral contract for the article generator. Controls output format (the JSON shape), tone, wikilink policy, and style rules.
Summary — system prompt
Used by Generate Summary on articles and by the re-index job to produce short, search-friendly blurbs.
Advanced prompts (user-message templates, labels, connection test)
Article generation — user preamble
First line of the user message sent to the model. Placeholder: {title}.
Article generation — guidance label
Heading inserted above any free-form guidance the user types into the generator dialog.
Summary — user template
User message for the summary generator. Placeholders: {title}, {content}.
Connection test — system prompt
Used by the Test Connection button above.
Connection test — user message
The model's reply is shown inline next to the Test Connection button.
Batch article — shared preamble
Prepended to the per-article guidance when generating a batch of articles together, so each call knows about its siblings. Placeholders: {count}, {allTitles}.
Timeline events — system prompt
Behavioral contract for the timeline-events generator. Controls the output JSON shape, year handling, tone, and the duplicate-avoidance rule.
Timeline events — user preamble
First lines of the user message sent to the timeline generator. Placeholders: {timelineName}, {count}.
