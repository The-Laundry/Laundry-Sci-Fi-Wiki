Help & Guide

A reference for using the Encyclopedia of Many Things.

🚀 Getting Started

Open a terminal in the encyclopedia/ folder and run:

python -m http.server 8000

Then open http://localhost:8000 in Chrome, Edge, or Arc. Click the No folder button in the top right to connect your data folder — pick the data/ subfolder inside your encyclopedia folder. The app will read and write real JSON files from that point on.

💡 You only need to connect the folder once per browser session. The browser remembers it until you close and reopen it.

✍️ The Article Editor

Click New Article in the top right, or Edit Article on any article page.

Toolbar
H1 / H2 / H3 — Headings (populate the Table of Contents)
B I U S — Bold, italic, underline, strikethrough
Align — Left, centre, right, justify
Lists — Ordered or unordered
Indent — Tab / Shift+Tab also work
xⁿ x₂ — Superscript / subscript
HR — Insert a horizontal rule
Wikilinks

Type [[Article Name]] anywhere to create a link.

Blue = article exists. Red = broken link (click to create the missing article).

Table of Contents

Auto-generated from H1/H2/H3 headings. Appears when there are 2+ headings.

📂 Article Manager

Nested categories — Unlimited depth. Use + Sub on any category to create a child.
Drag & drop — Drag article rows onto category headers or the Uncategorized zone to move them.
Rename — Click Rename, type, press Enter.
Wikibox Templates — Define reusable field layouts at the bottom of the manager page. They appear in the template dropdown in any editor.

📅 Timelines

Multiple timelines — Each is its own page. Events can belong to any number of timelines.
Calendar selector — Switch between five in-universe calendars. All dates are stored in CE internally and converted on the fly.
Importance levels — Insignificant (dot only) → Trivial → Minor → Notable (default) → Major → Milestone. Controls card size and visual weight.
Eras — Coloured date ranges that appear as bands on the timeline and segments in the minimap. Click an era label to edit it.
Minimap — The narrow bar on the right. Click anywhere to jump. Era names are clickable snap-to buttons.
Date override — Set a manual start/end year on a timeline to extend it beyond your earliest/latest event.

💾 Saving & Backup

With a folder connected, everything saves automatically to .json files. To move your wiki to another device, copy the entire encyclopedia/ folder and connect the data/ subfolder on the new device.

Use Data → Export All Data for a single-file JSON backup at any time.

⚠️ Without a connected folder, data lives in browser storage and can be erased if you clear your browser. Export regularly.

🌐 GitHub Pages Hosting

Push the encyclopedia/ folder to a GitHub repository and enable Pages in Settings. The wiki will be live at https://username.github.io/repo-name.

GitHub Pages is read-only, so the File System API won't be available there — the app falls back to localStorage. Use it as a read/share view of a JSON import, not as a live editing environment.
